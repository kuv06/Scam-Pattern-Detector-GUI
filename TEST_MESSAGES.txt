QUICK REFERENCE: EXPECTED DIALOGS BY MESSAGE TYPE
================================================================================

CRITICAL/HIGH Message Risk:
  → Shows RED/ORANGE risk panel
  → Shows "High Risk Message" warning dialog (WARNING icon)
  → Shows URL warnings grouped by risk

MEDIUM Message Risk:
  → Shows YELLOW risk panel
  → Shows "Medium Risk Message" warning dialog (WARNING icon)
  → Shows URL warnings grouped by risk

LOW Message Risk:
  → Shows BLUE risk panel
  → Shows "Low Risk Message" warning dialog (INFO icon)
  → Shows URL warnings if any URLs found

NO SCAM DETECTED:
  → Shows GREEN risk panel
  → No warning dialog
  → Shows "No obvious scam patterns found" message

================================================================================
TESTING TIPS
================================================================================

1. Copy a test message above and paste it into the input area.
2. Click "Scan Message" to scan for patterns + URLs (shows all warnings).
3. Or click "Check URL" to analyze only URLs found in the message.
4. Check the Risk Level (colored panel) matches expected level.
5. Verify dialogs appear with correct grouping of URLs by risk.
6. Click "Clear" to reset and try the next message.
7. Try sorting results by Category, Length, etc. to verify sorting works.

Expected Behavior Summary:
  ✓ HIGH risk URLs (shorteners, @, IPs) appear under "HIGH:" in dialog
  ✓ MEDIUM risk URLs (many dashes, long) appear under "MEDIUM:" in dialog
  ✓ LOW risk URLs (keywords, unusual TLDs) appear under "LOW:" in dialog
  ✓ Text patterns are displayed in "OTHER DETECTED INDICATORS"
  ✓ Overall message risk shown in colored panel + warning dialog
  ✓ Both "Scan Message" and "Check URL" buttons trigger warnings
  ✓ No warnings if no scam patterns detected (green "NO SCAM DETECTED" panel)
